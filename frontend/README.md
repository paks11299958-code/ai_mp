# 핸드폰에서 AI 채팅 앱 실행하는 방법

이 앱은 웹 브라우저에서 동작하는 웹 애플리케이션(SPA)입니다. 핸드폰에서 실행하려면 이 파일들이 웹 서버를 통해 호스팅되어야 합니다. 다음 두 가지 방법 중 하나를 선택하세요.

## 방법 1: 같은 Wi-Fi를 이용한 로컬 접속 (테스트용, 가장 쉬움)

PC와 핸드폰이 **같은 Wi-Fi**에 연결되어 있을 때 사용할 수 있는 방법입니다.

1. **PC에서 로컬 웹 서버 실행:**
   - **VS Code 사용자:** `Live Server` 확장 프로그램을 설치하고 `index.html` 파일에서 우클릭 후 `Open with Live Server`를 클릭합니다.
   - **Node.js 사용자:** 터미널에서 `npx http-server` 명령어를 실행합니다.
   - **Python 사용자:** 터미널에서 `python -m http.server` 명령어를 실행합니다.
2. **PC의 내부 IP 주소 확인:**
   - 윈도우: 명령 프롬프트(cmd)에서 `ipconfig` 입력 후 'IPv4 주소' 확인 (예: `192.168.0.15`)
   - 맥: 터미널에서 `ifconfig | grep inet` 입력 후 확인
3. **핸드폰에서 접속:**
   - 핸드폰의 웹 브라우저(크롬, 사파리 등)를 열고 주소창에 `http://[PC의 IP 주소]:[포트번호]`를 입력합니다.
   - 예: `http://192.168.0.15:5500` 또는 `http://192.168.0.15:8080`

## 방법 2: 무료 웹 호스팅 서비스 이용 (외부 접속 가능, 배포용)

인터넷이 연결된 어디서나 핸드폰으로 접속하려면 웹 호스팅 서비스에 배포해야 합니다.

1. **Vercel, Netlify, GitHub Pages** 등의 무료 정적 웹 호스팅 서비스에 가입합니다.
2. 현재 작성된 모든 파일(`index.html`, `App.tsx`, `types.ts` 등)을 해당 서비스에 업로드하거나 GitHub 저장소와 연동합니다.
3. **중요 (API 키 설정):** 
   - 현재 코드는 `process.env.API_KEY`를 통해 Gemini API 키를 불러옵니다.
   - 호스팅 서비스(Vercel, Netlify 등)의 설정(Settings) -> 환경 변수(Environment Variables) 메뉴로 이동하여 `API_KEY`라는 이름으로 본인의 Gemini API 키 값을 등록해야 합니다.
   - *참고: 현재 코드는 번들러(Vite, Webpack 등) 없이 브라우저에서 직접 ESM을 사용하는 방식이므로, 실제 상용 배포 시에는 Vite 등을 사용하여 빌드 과정을 거치는 것이 보안 및 성능에 좋습니다.*
4. 배포가 완료되면 제공된 URL(예: `https://my-ai-chat.vercel.app`)로 핸드폰 브라우저에서 접속합니다.

## 팁: 핸드폰 앱처럼 사용하기 (홈 화면에 추가)
핸드폰 브라우저로 접속한 뒤, 브라우저 메뉴에서 **"홈 화면에 추가"** (크롬) 또는 **"홈 화면에 추가"** (사파리)를 선택하면 일반 앱처럼 아이콘이 생성되고 전체 화면으로 실행할 수 있습니다.
